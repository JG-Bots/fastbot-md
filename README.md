# Comandos para instalar o bot!

```
pkg upgrade -y
pkg update -y
termux-setup-storage
pkg install nodejs -y
pkg install nodejs-lts -y
pkg install git -y
pkg install ffmpeg -y

cd /sdcard
git clone https://github.com/JG-Bots/fastbot-md
cd fastbot-md
sh start.sh
```
